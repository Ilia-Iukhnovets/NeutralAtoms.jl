# Examples

The package is easiest to understand in the same order as a neutral-atom
experiment: characterize the trap, infer temperature, model single-atom
Rydberg excitation, and finally move to blockade-mediated two-qubit dynamics.

These example pages use lightweight parameters so they build quickly while still
exercising the current dict-based laser parameter API.

- [Release And Recapture](release_recapture.md)
- [Single-Atom Two-Photon Rydberg Dynamics](single_atom_rydberg.md)
- [Two-Qubit Blockade Simulation](two_qubit_cz.md)
